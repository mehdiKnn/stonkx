
## About StonkX project

StonkX is a laravel powered sneakers e-commerce.

## Install
 
 First you need to install Laravel ressources
```
composer install
````
Laravel front end scaffolding

```
npm install
```

## Database

First create the DB
```
php artisan migrate
```

Then fill the DB with datas
```
php artisan db:seed
```

## Run the project

```
php artisan run serve
```
